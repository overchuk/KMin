К оглавлению

Все классы системы произведены из единого предка, KM_. Класс имеет единственное поле - $data, ассоциативный массив. Все данные хранятся в этом массиве, и механизм сохранения и восстановления объектов класса реализует сохранение типа объекта и значения $data.

KM_ не предполагает создание объектов, и как правило должен быть использован только для наследия дугих классов. Управление классами производится при помощи namespace KMclass

$object->data['kmin']['class'] - Имя класса данного объекта, без префикса KM_
$objcet->_init($default, $option) - Инициализирует данные $data получая на вход массив значений поумолчанию, и массив переопределений умолчальных значений.
$objcet->name() - Возвращает имя класса объекта.
